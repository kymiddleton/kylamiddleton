Volunteer Mommy

Inspiration

As the chair for Parents-with-Preschoolers I have the responsibility of managing 150+ volunteers. Managing paper sign-up sheets, scheduling events and assigning volunteers in spreadsheets prompted the idea of an application to simplify and automate the process.

Overview

Upon arrival to the website new users are prompted to create an account and verify their credentials by loging in to the main site. Authenticated users have access to the dashboard area where users have the ability to add adult volunteers and add enrolled children.

Accomplishments

This was an independent final project for an immersive web development course using the MERN stack.

  • MongoDB
  • Express
  • React

  • HTML5
  • CSS
  • Semantic UI
  • Material UI
  • Axios
  • Json Web Token
